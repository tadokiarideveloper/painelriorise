RIO RISE - Painel Administrativo

Arquivos do site:
- index.html
- styles.css
- app.js
- assets/logo-rio-rise.jpg

Como usar:
1. Abra o index.html no navegador para testar.
2. Para publicar, suba a pasta inteira no Cloudflare Pages, Netlify, Vercel ou hospedagem estática.
3. O login já está configurado com o usuário e senha que você informou. Eles não aparecem na tela, e a senha não está salva em texto puro no código.
4. Os registros ficam salvos no navegador usando localStorage.
5. A evidência agora é somente por link. A função de anexar vídeo/arquivo foi removida.
6. Na visualização da ficha, o link de evidência aparece clicável e também com botão para abrir.
7. O botão de copiar link gera uma ficha pública somente daquele registro.
8. O layout foi ajustado para mobile.
9. Use Exportar planilha .xlsx para baixar uma planilha .xls com resumo, meta, tipos de punição, registros completos, links de evidência e links das fichas.
10. A opção Importar backup continua aceitando backups antigos em JSON.

Aviso importante:
Por ser um site estático, ele serve para uso pessoal/privado. Para segurança profissional real com banco online, permissões e armazenamento compartilhado entre aparelhos, precisa de backend com servidor e banco de dados.

Atualização da planilha:
- O botão Exportar planilha .xlsx agora gera um arquivo .xlsx real.
- A planilha possui abas Resumo, Registros, Resumo por Tipo e Resumo por Jogador.
- A aba Registros vem em formato de tabela com cabeçalho fixo, filtro e links clicáveis para evidência e ficha.
